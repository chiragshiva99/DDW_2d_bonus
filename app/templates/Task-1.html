Task 1

We found a Covid-19 Dataset which has data for the number of covid cases, deaths, vaccination rate etc across many countries. This data set was created by Our World in Data, which relies on the work of scholars worldwide to compile and display data in an understandable way. It is a live file that is constantly being updated as countries report their statistics day by day, which gives us a bird’s eye view of not only how the situation has evolved in different countries, but also how the situation has evolved over time. However, it is limited by whether countries choose to report their data, whether it is accessible, and how comprehensive they are.

Having looked through the entire covid situation, we thought we should analyse the month where many countries had a surge in covid cases. Thereby, using filtered data for June 2020 where we could sum up the values for the daily cases and deaths for that particular month.

We decided to run all our variables through python to plot ‘Sum of deaths’ in June against all the other variables individually to see which variables had a strong correlation to sum of deaths. As a result, we ended up with ‘Population Density’, ‘Monthly Cases’, ‘Human development index’, ‘GDP per capita’ and ‘Median Age’. After trying different combinations of 3-4 variables within excel, we came to a conclusion to use ‘Population Density’, ‘Monthly Cases’ and ‘Human Development Index’ as it gives us the best adjusted R squared value of 0.788  as shown in the excel.
